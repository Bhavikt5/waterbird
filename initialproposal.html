SaaS Platform Overview
Project Quotation SaaS
for Proposal Engineers
A cloud-based platform that helps proposal engineers create technical and commercial project quotations for water and wastewater treatment plants — faster, standardised, and error-free.
Water treatment Wastewater treatment Multi-office SaaS
What this is not
Not a website · Not a landing page · Not a document template · Not an Excel sheet
It is a fully functional quotation software system.
Who it's for
The Proposal Engineer's workflow today
Creating a quotation manually is slow, error-prone, and inconsistent across offices. This platform replaces that process.
Proposal engineer
Creates quotations, enters parameters, selects equipment, submits for approval
Manager / Approver
Reviews, approves, or returns quotations before they reach the client
Admin
Manages users, equipment catalog, pricing master, templates, margins, tax
Multi-office access
Teams in Dubai, Abu Dhabi, India and other branches use the same system
How it works
End-to-end user flow
Log in
Secure cloud login from any office
Create new quotation
Unique quotation number, revision number, project record generated automatically
Enter 50+ parameters
Client info, plant type, flow rate, water quality, treatment scope, commercial terms
Equipment & BOQ generation

Auto-costing
Equipment, fabrication, civil, electrical, installation, margin, tax — all calculated
Submit for approval
Manager reviews and approves or returns for correction
Download & deliver
Generate PDF, Word, or Excel quotation — ready for the client
